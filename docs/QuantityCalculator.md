# DeepBook V3 数量计算器

这是一个演示 DeepBook V3 SDK 中三个重要数量计算方法的交互式工具。

## 功能特性

### 支持的方法

1. **getQuoteQuantityOut**
   - 输入基础资产数量，计算能获得的报价资产数量
   - 适用于"卖出"场景
   - 例如：输入 100 SUI，计算能获得多少 USDC

2. **getBaseQuantityOut**
   - 输入报价资产数量，计算能获得的基础资产数量
   - 适用于"买入"场景
   - 例如：输入 1000 USDC，计算能获得多少 SUI

3. **getQuantityOut**
   - 通用转换方法，支持双向计算
   - 只能有一个输入非零
   - 更灵活的计算选项

### 支持的交易池

- **SUI/USDC**: SUI 与 USDC 交易对
- **DEEP/SUI**: DEEP 与 SUI 交易对  
- **SUI/USDT**: SUI 与 USDT 交易对

## 使用方法

1. **选择交易池**: 点击想要计算的交易池卡片
2. **选择计算方法**: 选择三种方法中的一种
3. **输入数量**: 根据选择的方法输入相应的数量
4. **查看结果**: 点击"计算预估结果"按钮获取计算结果

## 结果说明

计算结果包含以下信息：

- **输入参数**: 显示你输入的原始数量
- **输出结果**: 
  - `baseOut`: 基础资产输出数量
  - `quoteOut`: 报价资产输出数量
- **费用信息**: 
  - `deepRequired`: 执行交易需要的 DEEP 代币数量
- **有效汇率**: 基于计算结果显示的实时汇率

## 重要说明

- 这些都是**只读操作**，不会执行实际交易
- 结果为**预估值**，实际交易可能因市场变化而有差异
- DEEP 代币用于支付交易费用
- 数量使用标准十进制格式（如 10.5）
- 如果网络连接失败，会显示模拟数据用于演示

## 技术实现

- 使用 `@mysten/deepbook-v3` SDK
- 连接到 Sui 测试网络
- Vue 3 + Composition API
- 响应式设计，支持移动端

## 访问方式

- 路由路径: `/deepbook/calculator`
- 导航栏: 其他 → QuantityCalculator

这个工具帮助开发者和用户更好地理解 DeepBook V3 的价格计算机制，为实际交易提供参考。
